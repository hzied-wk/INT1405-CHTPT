# INT1405 - Các hệ thống phân tán

Kho lưu trữ này chứa tài liệu học tập, bài tập ôn tập, giáo trình và hình ảnh minh họa cho môn học **INT1405 - Các hệ thống phân tán** tại trường đại học.

## 📚 Nội dung chính

Repo được tổ chức thành các thư mục sau:

- **Đề ôn tập chương**  
  Chứa các file Word (.docx) tổng hợp câu hỏi và bài tập ôn tập theo từng chương:
  - `bài tập ôn tập chương 123.docx`
  - `Câu hỏi ôn tập chương 4.docx`
  - `Câu hỏi ôn tập chương 12.docx`
  - `Cau_hoi_bai_tap_chuong_5_6.docx`
  - `ôn tập.docx`

- **Giáo trình**  
  Bao gồm các tài liệu PDF phục vụ học tập:
  - `276909618-Bai-giang-he-thong-phan-tan.pdf`
  - `Distributed_Systems_3.pdf`
  - `Giao trinh Cac He thong phan tan 2024 16-11-20.pdf`

- **Thư mục hình ảnh**  
  Chứa sơ đồ, biểu đồ và hình minh họa:
  - `sơ đồ class.png`
  - `sơ đồ kiến trúc tổng quan.png`
  - `sơ đồ tuần tự 1.png`
  - `sơ đồ tuần tự 2.png`
  - `usecase tổng quan.jpg`

- **README.md**  
  Tài liệu giới thiệu repo.

## Mục tiêu

- Hỗ trợ sinh viên ôn tập và chuẩn bị cho kỳ thi môn **Các hệ thống phân tán**.  
- Cung cấp tài liệu tham khảo (giáo trình, bài giảng, sơ đồ minh họa).  
- Tạo môi trường chia sẻ kiến thức và hợp tác giữa các thành viên.

## Cách sử dụng

1. Clone repo về máy:
   ```bash
   git clone https://github.com/hzied-wk/INT1405-CHTPT.git
   ```
2. Mở bằng VS Code hoặc bất kỳ IDE nào để truy cập tài liệu.  
3. Tham khảo các thư mục theo nhu cầu:
   - Ôn tập → **Đề ôn tập chương**
   - Lý thuyết → **Giáo trình**
   - Hình minh họa → **Thư mục hình ảnh**
